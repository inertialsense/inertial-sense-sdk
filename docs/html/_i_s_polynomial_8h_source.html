51// ----------------------------------------------------------------------------
52// y = horner(degree, coef, x) evaluates a polynomial y = f(x) at x. The polynomial has
53// degree = n-1. The coefficients of the polynomial are stored
54// in the 1-D array c, which has n elements.
55//
56// NOTE: The polynomial coefficients are multipliers of monomial terms of
57// decreasing order. In other words, the polynomial is assumed to be
58// written in the form
59//
60// y(x) = coef[0]*x^n + coef[1]*x^(n-1) + ... + coef[n-1]*x + coef[n]
61//
62// where n is the order (coef_size-1) of the polynomial and the largest index in coef is n.
63float ixPolyHorner(const int coef_size, const float coef[], const float x);
