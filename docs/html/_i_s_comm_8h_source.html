151 /*
152 Packet Overview
153 
154 Byte
155 0 Packet start byte
156 1 Packet indo: ID (mask 0x1F) | reserved bits (mask 0xE)
157 2 Packet counter (for ACK and retry)
158 3 Packet flags
159 
160 // packet body, may or may not exist depending on packet id - packet body is made up of 4 byte or 8 byte values.
161 4-7 Data identifier
162 8-11 Data length
163 12-15 Data offset
164 16-19 Data start
165 (n-8)-(n-5) Last piece of data
166 // end data
167 
168 n-4 Reserved
169 n-3 Checksum high byte
170 n-2 Checksum low byte
171 n-1 Packet end byte
172 */
