157/*
158Packet Overview
159
160Byte
1610 Packet start byte
1621 Packet indo: ID (mask 0x1F) | reserved bits (mask 0xE)
1632 Packet counter (for ACK and retry)
1643 Packet flags
165
166// packet body, may or may not exist depending on packet id - packet body is made up of 4 byte or 8 byte values.
1674-7 Data identifier
1688-11 Data length
16912-15 Data offset
17016-19 Data start
171(n-8)-(n-5) Last piece of data
172// end data
173
174n-4 Reserved
175n-3 Checksum high byte
176n-2 Checksum low byte
177n-1 Packet end byte
178*/
